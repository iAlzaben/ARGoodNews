# 🌟 خبر سار - Khabar Sar

موقع عربي تفاعلي حديث ومبتكر مخصص بالكامل لـ **الأخبار الإيجابية والأخبار السارة**. تم تصميم هذا الموقع ليكون بمثابة **تجربة بصرية وحسية مهدئة ومحفزة للأمل**، مبتعداً تماماً عن رتابة المواقع الإخبارية التقليدية.

---

## ✨ المميزات الرئيسية للموقع:
1. **بوابة التفاؤل (Splash Portal):** واجهة ترحيبية ناعمة بنور الشمس الدافئ ترحب بالزوار بعبارات مبهجة تتغير بناءً على وقت زيارتك (صباح/مساء)، وتنساب بسلاسة إلى المحتوى عند بدء التصفح.
2. **مرشح المزاج والنبض الإيجابي (Mood Filter):** لتصفية الأخبار السارة بناءً على "المشاعر" التي تثيرها في القلب:
   * **مُلهم ✨** (قصص الكفاح والذكاء الاصطناعي والإرادة)
   * **يبعث على الأمل 🌱** (مشاريع البيئة والزراعة الصحراوية الحديثة)
   * **يرسم البسمة ❤️** (المبادرات الإنسانية ومساعدة المحتاجين)
   * **فخر عربي 👑** (الإنجازات والابتكارات بأيدي شبابنا العربي)
3. **جرعة التفاؤل اليومية (Daily Dose of Joy):** أداة مركزية تفاعلية. عند النقر عليها، تنطلق مؤثرات برّاقة (تأثيرات جزيئية ملونة على Canvas) وتكشف عن حكمة مأثورة أو حقيقة علمية تسعد يومك.
4. **فوانيس الأمل (Hope Lanterns Guestbook):** لوح تفاعلي يتيح لك كتابة وإرسال خبر سعيد أو كلمة طيبة، لتشاهد فانوسك المضيء يصعد حياً ويحلق في فضاء الموقع حاملاً اسمك وكلماتك!
5. **خريطة الأمل التفاعلية (Interactive Map of Joy):** خريطة مبتكرة ومبسطة للوطن العربي تحتوي على نقاط مضيئة ونابضة بالحياة؛ عند المرور عليها تكشف عن مبادرات ومشاريع واعدة جارية في تلك البقعة.
6. **مؤثرات صوتية ومرئية مهدئة:** دمج مؤثرات صوتية ناعمة عند النقر والتفاعل (مع زر كتم/تفعيل الصوت)، ونظام انتقالي فائق النعومة بين مظهر الشروق (Light Mode) والواحة الليلية (Dark Mode).

---

## 🎨 الخطوط والجمالية:
تم إعداد الموقع ليعتمد بالكامل على **خط ثمانية (Thmanyah Font)** ليعطي طابعاً حصرياً وعصرياً في غاية الرقي والفخامة:
* **Thmanyah Serif Display:** للعناوين والمانشيتات الإخبارية الفخمة.
* **Thmanyah Serif Text:** لنصوص الأخبار ومقاطع القراءة.
* **Thmanyah Sans:** للأزرار والواجهات التفاعلية.

### 📥 كيفية تفعيل خط ثمانية (Thmanyah Font):
1. قم بزيارة موقع خط ثمانية الرسمي: [font.thmanyah.com](https://font.thmanyah.com).
2. قم بتحميل ملفات الخط (يفضل بصيغة `WOFF` أو `WOFF2` أو `TTF`).
3. انقل الملفات بعد تحميلها وتسميتها إلى المجلد التالي في المشروع:
   `assets/fonts/`
   لتصبح المسارات كالتالي:
   * `assets/fonts/Thmanyah-Serif-Display.woff2`
   * `assets/fonts/Thmanyah-Serif-Text.woff2`
   * `assets/fonts/Thmanyah-Sans.woff2`
4. **💡 ملاحظة هامة:** قمنا بدمج خطوط بديلة في منتهى الجمال من قوقل مثل (**Amiri** للعناوين و **Cairo** للنصوص) كخيار احتياطي ممتاز، ليعمل الموقع بأعلى درجات الأناقة فوراً وبشكل تلقائي حتى قبل قيامك بتحميل ملفات الخطوط!

## 🚀 كيفية تشغيل الموقع محلياً:
الموقع مبني بالكامل بأحدث معايير الويب الفائقة السرعة والخفيفة بدون الحاجة لأدوات تشغيل معقدة أو عمليات تثبيت مطولة.

### الطريقة الأبسط:
فقط قم بفتح ملف `index.html` مباشرة في أي متصفح ويب حديث لتستمتع بالتجربة الكاملة!

### تشغيله كخادم محلي (مستحسن لتجربة الصوتيات والتأثيرات كاملة):
1. إذا كان لديك محرر الكود (VS Code)، يمكنك استخدام إضافة **Live Server**.
2. أو من خلال فتح سطر الأوامر في مجلد المشروع وتشغيل خادم محلي سريع عبر Node.js:
   ```bash
   npx live-server
   ```
   أو عبر بايثون:
   ```bash
   python -m http.server 8000
   ```
   ثم فتح المتصفح على الرابط: `http://localhost:8000`.
